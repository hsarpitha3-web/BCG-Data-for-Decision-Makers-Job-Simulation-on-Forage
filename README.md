# BCG-Data-for-Decision-Makers-Job-Simulation-on-Forage

## 📌 Overview
This project is part of the BCG Data for Decision Makers Job Simulation on Forage, focused on analyzing marketing performance and providing data-driven recommendations to a hypothetical client.

## 🔍 Key Tasks
- Analyzed multi-channel campaign data using simulated datasets  
- Evaluated customer conversion trends across channels and audiences  
- Assessed campaign effectiveness and performance gaps  
- Delivered clear, data-backed recommendations aligned with business goals  

## 🛠 Tools Used
- Microsoft Excel  
- Data Analysis & Visualization techniques  

## 💡 Key Learnings
- Marketing analytics and campaign performance evaluation  
- Customer conversion and funnel analysis  
- Translating data into actionable business insights  
- Data storytelling and recommendation building
